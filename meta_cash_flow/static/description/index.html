Cash Flow Statement

This module allows creating Cash Flow Statement on account tag ( Operating Activities, Financial Activities and Investing and Extraordinary Activities).

Dependencies
  • account.
  • custom_report.
Quick View
  • App create a menu named 'Cash Flow Statement' Under 'Accounting/Reports'.
  • To get proper report, account needed to tagged correctly.
  • On wizard, date range and move line filter is available
  • Report have both html and pdf view.
